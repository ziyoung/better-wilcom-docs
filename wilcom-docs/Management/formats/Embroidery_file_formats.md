# Embroidery file formats

There are two types of embroidery file format:

| File type | Description                                                                                                         |
| --------- | ------------------------------------------------------------------------------------------------------------------- |
| Design    | Design files usually contain digitized shapes and lines, selected stitch types and stitch values and effects.       |
| Machine   | Machine files generally contain stitches and machine functions and are only suited to specific embroidery machines. |

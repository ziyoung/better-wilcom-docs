# Working with object properties

When you start a new design, EmbroideryStudio uses the default settings associated with the chosen [fabric](../../glossary/glossary) (if activated). Other property settings are stored with the design [template](../../glossary/glossary). You can override most settings on an object-by-object basis. For this reason, EmbroideryStudio distinguishes three sets of object properties â€“ default, actual and current.
